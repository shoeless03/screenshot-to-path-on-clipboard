1. click print screen
2. overlay appears
3. select region
4. screenshot saved
5. path to file copied to clipboard

that's it.

## Install

```
git clone https://github.com/shoeless03/screenshot-to-path-on-clipboard.git
cd screenshot-to-path-on-clipboard
pip install -r requirements.txt
python main.py
```

Requires Python 3.8+. Runs as a system tray app — right-click the tray icon for settings.
